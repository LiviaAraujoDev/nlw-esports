# NLW Esports
O flix é um projeto que exibir um filme aleatório, por meio dos dados fornecidos pela API do themoviedb.org<br><br>

![GitHub Logo](interface-web.png)

## Tecnologias

Esse projeto foi desenvolvido com as seguintes tecnologias:

### Frontend Web
- TypeScript
- Tailwind CSS
- React
- Vite<br><br>

### Backend
- TypeScript
- Node
- SQLite
- Prisma

### Mobile
- React Native
- Typescript<br><br>

## Run Locally

~~~bash
$ npm run dev
~~~
